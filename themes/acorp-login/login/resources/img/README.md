This directory contains image assets for the A-Corp Keycloak login theme.

## Production Logo

For production, copy the official website emblem from the A-Corp website repository:

```text
public/assets/alvestrasza-emblem-transparent.png
```

into this directory:

```text
themes/acorp-login/login/resources/img/alvestrasza-emblem-transparent.png
```

Then update `--acorp-emblem` in `../css/acorp-login.css` if needed.
